Cryptography Program






About



Developed a cipher that encrypts and decrypts messages containing any character in the alphabet. I will be adding digits to the cipher's valid input range. User can put in a string that he/she wants to encrpyt, the program will prompt for a shift which is a number and is used in the algorithm to encrpyt the string. The user can send that encrypted message & the specific shift used to another user who has access to this program. The program then is able to decrypt the message using that information.




Language/Technology Used



C++




The Logic



To encrypt a string, an array is used to keep the alphabet stored. Then the program calculates the indices of the user inputted string according to the alphabet array. Each character in the string then goes into a for-loop where the shift is used to create a unique number which is added to the original index of the characters and a new character is picked out corresponding to the new index calculated.

Similar concept is used to decrypt the string. The program calculates the indices of the string. Then the program uses the unique shift given and comes up with unique number used for each character and instead of adding, the program subtracts that unique number from the calculated indices revealing the original message
